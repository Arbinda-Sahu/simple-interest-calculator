# Simple Interest Calculator

This project is a simple interest calculator that calculates the interest based on principal, rate, and time.

## Formula
Simple Interest = (Principal × Rate × Time) / 100

## Example
If Principal = 1000  
Rate = 5%  
Time = 2 years  

Interest = (1000 × 5 × 2) / 100 = 100
